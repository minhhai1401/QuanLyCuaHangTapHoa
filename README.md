# Quản Lý Cửa Hàng Tạp Hóa

Dự án quản lý cửa hàng tạp hóa được xây dựng bằng **ASP.NET + SQL Server**.  
Ứng dụng hỗ trợ quản lý sản phẩm, hóa đơn, khách hàng và thống kê doanh thu.

## Chức năng chính
- Đăng nhập / Đăng xuất
- Quản lý sản phẩm
- Quản lý khách hàng
- Quản lý hóa đơn
- Thống kê doanh thu
